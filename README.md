jwalker
----

Parses a file stream for JSON and pretty prints it.

# Usage

    Usage: jwalker [options]
    
      Use -- (or omit) for stdin or stdout.
    
      Options:
        -i=[file], --input=[file]         Input file.
        -o=[file], --output=[file]        Output file.
        -e, --exit                        Exit on errors.
